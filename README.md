# App Android – Navegação entre Telas

Aplicativo Android simples desenvolvido em Kotlin com o objetivo de praticar conceitos fundamentais do desenvolvimento Android, como navegação entre Activities, criação de layouts em XML e organização básica de projeto.

## Descrição:

Este projeto consiste em um aplicativo com duas telas (Activities).
A tela inicial possui um botão que navega para a segunda tela, enquanto a segunda tela contém um botão que retorna à tela inicial.
Ambas as telas utilizam layouts XML e um fundo personalizado para fins de aprendizado de design básico.

## Funcionalidades:

  - Navegação entre duas Activities

  - Botão para avançar para a segunda tela

  - Botão para retornar à tela inicial

  - Layouts desenvolvidos em XML

  - Aplicação de fundo personalizado nas telas
    
## Tecnologias Utilizadas:

  - Kotlin

  - Android Studio

  - XML

  - ConstraintLayout

  - LinearLayout

## Estrutura do Projeto:

  - MainActivity: tela inicial com botão de navegação

  - SecondActivity: tela secundária com botão de retorno

  - Layouts separados para cada Activity

  - Recursos visuais organizados na pasta res

## Objetivo do Projeto:

Projeto desenvolvido com fins educacionais, visando consolidar conhecimentos iniciais em desenvolvimento Android nativo e controle de navegação entre telas.
